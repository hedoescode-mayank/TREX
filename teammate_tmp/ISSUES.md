# Proposed GitHub Issues for T.R.E.X Beta

Copy and paste these into your GitHub repository's "Issues" tab.

---

## 1. [Feature] PDF Parsing Support for Resume AI
**Description**: 
Currently, the Resume AI requires users to paste text manually. We need to implement direct PDF and DOCX file upload and parsing using libraries like `pypdf` on the backend and handling the file upload on the frontend.
**Label**: `enhancement`, `priority: high`

---

## 2. [Feature] Interactive Career Roadmap Generation
**Description**: 
Develop the 'Career AI' module to generate visual, step-by-step career path roadmaps based on a user's target role and current skills.
**Label**: `new-module`, `beta`

---

## 3. [UI/UX] Implement High-Fidelity Dark Mode
**Description**: 
The initial focus was on the premium glassy light theme. We need to add a toggle and full CSS support for a sleek Dark Mode that maintains the "T.R.E.X" premium aesthetic.
**Label**: `ui/ux`, `enhancement`

---

## 4. [Optimization] Real-time Cost of Living Data Integration
**Description**: 
Move from static JSON-based city data to fetching real-time data from external APIs (like Numbeo or Google Places) to keep the cost analyzer accurate.
**Label**: `data`, `backend`

---

## 5. [Bug] Mobile Response Polish for Analysis Charts
**Description**: 
The Recharts components in the City Analyzer needs better responsive configurations to ensure they look great on all screen sizes.
**Label**: `bug`, `ui/ux`
